## 📊Supermarket Sales Dashboard (Power BI) 

🚀 An interactive Power BI dashboard to analyze supermarket sales performance, customer behavior, and revenue trends.

---

## 🎯 Objective

This project aims to transform raw transactional data into meaningful insights that help businesses optimize sales strategies and improve profitability.

---

## 🔍 Key Insights

* 💰 Total revenue reached **$307.59K**, with total profit of **$15.38K** (~5% margin), indicating room for margin improvement
* 🛍️ **Fashion accessories** achieved the highest average rating, suggesting strong customer satisfaction
* 👥 Customer distribution is relatively balanced across **gender and membership types**, with no dominant segment
* 💳 Payment methods (**cash, e-wallet, credit card**) show nearly equal usage
* 📈 Sales trend fluctuates across months, highlighting opportunities for seasonal promotions
